# 🎯 Ultra Resume Builder — Claude AI Skill

> Generate ATS-optimized (95+ score), tailored resumes for any job 
> description in minutes. Built as a reusable Claude skill.

---

## What It Does

Drop in a job description. Get back a resume that:
- Scores **95+/100 on ATS** keyword matching
- Is **tailored to the specific company** (deep research built-in)
- Surfaces experiences you forgot to include (branching interview)
- Outputs **MD + DOCX + PDF + Interview Prep report**
- Recommends **relevant certifications** with JD context
- Handles **3–5 jobs in batch mode** with shared discovery

Zero fabrication. Maximum reframing of real experience.

## How to Use

1. Open Claude (claude.ai)
2. Upload or paste `SKILL.md`
3. Say: *"Use this skill. Here's my resume and job description: [paste]"*
4. Answer the discovery questions
5. Download your DOCX + PDF
