# Research Prompts — Deep Company & Role Analysis

## Overview

Structured research templates for analyzing companies, roles, and success profiles.
Run these searches before building any resume section.

---

## Template A: Company Culture Research

### Search Queries to Run
```
1. "[Company name] company culture values 2024 2025"
2. "[Company name] CEO letter to employees OR CEO blog OR annual report"
3. "[Company name] glassdoor reviews culture"
4. "[Company name] recent news layoffs expansion product launch"
5. "[Company name] [department] team LinkedIn"
```

## Template C: Industry Benchmarking

### Search Queries
```
1. "Top [job title] resume examples [industry] 2024"
2. "[Industry] [job function] skills in demand 2025"
3. "[Job title] LinkedIn profiles [company name]"
4. "[Company name] [team] case study OR success story"
```

### Extraction Template

```
INDUSTRY BENCHMARKING
═══════════════════════════════════════
Industry: _______________
Function: _______________

WHAT TOP CANDIDATES IN THIS ROLE HAVE:
(Based on LinkedIn profiles + resume examples)
→ Average certifications: _______________
→ Common past companies: _______________
→ Typical education: _______________
→ Side projects / extracurriculars: _______________

DOMAIN TERMINOLOGY (use these naturally):
Finance:    DCF, EBITDA, LBO, IRR, NAV, AUM, yield curve, spread, NPA
Banking:    CRAR, NIM, GNPA, PCR, SLR, CRR, priority sector, MSME lending
FinTech:    API banking, UPI stack, digital lending, embedded finance
Marketing:  funnel optimization, attribution modeling, cohort analysis
Consulting: hypothesis-driven, workstream, deliverable, deck, synthesis
Tech PM:    sprint, backlog, OKR, epic, user story, A/B test, GTM

LANGUAGE CALIBRATION:
→ Avoid if under-qualified: "led 50-person team", "P&L ownership"
→ Use if entry-level: "contributed to", "supported", "analyzed", "built"
→ Use if mid-level: "managed", "designed", "owned", "delivered"
→ Use if senior: "drove", "scaled", "strategized", "transformed"

ATS WATCH-OUT KEYWORDS:
(Terms that trigger rejection if NOT in resume)
→ _______________
→ _______________
```

## Research Quality Checklist

Before proceeding to STEP 3 (ATS Extraction), confirm:

```
□ Company culture keywords identified (at least 5)
□ JD hard skills fully listed
□ JD action verbs noted
□ Industry terminology compiled
□ Seniority language calibrated
□ Recent company news checked
□ Success profile articulated

RESEARCH CONFIDENCE: Low / Medium / High
If Low → Note in resume that research was limited; focus on JD-only
If Medium → Use what's found + JD analysis
If High → Full personalization in summary + bullets
```
